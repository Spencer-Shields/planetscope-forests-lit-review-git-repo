# PlanetScope forestry literature review
This is the data and code used for the literature review:

Shields, S. G., Coops, N. C., Achim, A., Hamelin, R. C., & Mulverhill, C. (2025). A review of PlanetScope CubeSats for forest monitoring. Science of Remote Sensing, 100314.
Data and R code for systematic literature review on the use of PlanetScope CubeSats for forest monitoring. https://doi.org/10.1016/j.srs.2025.100314.

The raw data on each article is contained in the MS Excel spreadsheet "review_list_8.xlsx". The .R file "lit review analysis and graphs" is used to query and summarize the data.
